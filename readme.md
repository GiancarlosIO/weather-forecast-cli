## Weather CLI
A simple CLI to get the weather forecast of a specific location.

### How to use
1. Go to https://www.weatherapi.com/, create an account and get an api-key
2. Put your api-key value in the file `.env` as `API_KEY=your-value`
3. Run using `go run .`
